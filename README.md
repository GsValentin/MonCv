# 📄 Mon CV App

Application iOS développée avec Swift dans Xcode pour créer un CV interactif et apprendre le développement mobile.

---

## 🚀 Objectif du projet

Ce projet a pour but de m’entraîner au développement iOS en Swift et à l’utilisation de Git et GitHub dans un environnement de travail réel.

Il me permet de pratiquer :
- la création d’interfaces avec SwiftUI / UIKit
- l’organisation d’un projet Xcode
- la gestion de versions avec Git
- la publication sur GitHub

---

## 🛠️ Technologies utilisées

- Swift
- Xcode
- Git
- GitHub

---

## 📚 Fonctionnalités

- Affichage des informations personnelles
- Présentation du parcours
- Liste des compétences
- Structure évolutive pour ajout de nouvelles sections

---

## 🎯 Objectif futur

- Ajouter une interface plus avancée
- Exporter le CV en PDF
- Améliorer le design UI/UX
- Ajouter des animations
